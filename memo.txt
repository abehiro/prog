1. Githubでリポジトリ作成
2. ローカルで git init
3. git ad file
4. git commit -m "comment"
5. git status
6. git remote add origin https://github.com/username/dir
7. git fetch (リモートにREADMEができてるかも)（vi でセーブ）
8. git status
9. git push -u origin master
